## Demo
Experience the project live by opening `index.html` in your web browser.  

You will see:
- A visually appealing underwater background with animated bubbles.
- Fish swimming across the screen with random colors.
- Hotspot buttons that show mysterious ocean facts in a modal.
- Interactive flip cards showcasing fascinating underwater locations.
- An Ocean Chatbot that provides interesting responses to ocean-related keywords.

## Features
- **Animated Background:** Bubble animations simulate the underwater environment.  
- **Wave Text Animation:** Heading text floats and moves in a wave-like animation for a lively effect.  
- **Swimming Fishes:** Fish shapes move across the screen with dynamically changing colors.  
- **Hotspot Facts:** Clickable buttons display interesting ocean facts in a modal with a subtle flash effect.  
- **Flip Cards:** Cards displaying aquatic wonders can be flipped to reveal detailed descriptions.  
- **Random Fish Generator:** Users can generate a new set of fish with randomized details.  
- **Ocean Chatbot:** Ask ocean-related queries and receive informative or fun responses in real-time.  
- **Responsive Design:** Works on various screen sizes, providing an immersive experience.
